Implemented an Artificial Intelligence model capable of playing heads up no limit Texas-Holdem. This is done using a type of reinforcement learning technique counterfactual regret minimization.


Since Poker is a game with incomplete information, the machine learning model trains itself by playing the game with itself, evaluating all the different decisions they could have taken, and decide on their ideal strategy on each specific game state based on the resulting outcome.
